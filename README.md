## ğŸ“ Deskripsi Aplikasi
Aplikasi **Quiz UTS** merupakan aplikasi kuis interaktif berbasis **Flutter** yang dibuat untuk memenuhi tugas **UTS Pemrograman Mobile 6**.  
Aplikasi ini memungkinkan pengguna untuk:
- Memasukkan nama sebelum memulai kuis.  
- Menjawab serangkaian pertanyaan pilihan ganda.  
- Melihat skor akhir setelah menyelesaikan semua pertanyaan.  
- Mengulangi kuis dari awal.  

Fitur utama aplikasi:
- Terdiri dari minimal **2 halaman (Home, Quiz, Result)**.  
- Menggunakan **Provider** untuk state management.  
- Desain **responsif** dengan `Responsive` class.  
- Menggunakan **aset gambar dan font kustom (Poppins)**.  
- Memiliki **widget reusable** seperti `QuestionCard`, `OptionTile`, dan `AppHeader`.
